VISION

프로젝트 설명

탐지 엔진, 관리자 페이지, AI 분석 서버를 연동해 유해 웹 접속 시도를 탐지·차단하고, 정책 변경 사항과 운영 로그를 관리할 수 있도록 구성한 시스템입니다. 관리자는 웹에서 정책을 수정하고, 변경된 정책은 엔진에 반영되며, 탐지 로그와 이력 데이터를 기반으로 운영 상태를 점검할 수 있도록 구현했습니다.

기술 스택

Python · Django · FastAPI · MySQL · Bootstrap

주요 내용
  • 관리자 페이지에서 정책 등록, 수정, 삭제 및 적용 여부 관리
  • 정책 변경 시 탐지 엔진 반영을 위한 연동 처리 구현
  • 탐지 로그, 정책 수정 이력, 삭제 이력 조회 기능 구성
  • AI 분석 결과를 운영 정책 관리 흐름과 연결

News Data Collection Pipeline

프로젝트 설명

뉴스 사이트에서 기사를 주기적으로 수집해 데이터베이스에 저장하고, 누적된 기사 데이터를 관리할 수 있도록 구성한 프로젝트입니다. 단순 크롤링에 그치지 않고, 반복 수집되는 데이터를 저장·관리하는 흐름까지 고려해 데이터 처리 구조를 구현했습니다.

주요 내용
  • 뉴스 기사 데이터 크롤링 및 수집 자동화

  • 중복 데이터 처리와 기사 정보 조회 구조 구성
  • 반복 수집 기준의 데이터 관리 흐름 구현
기술 스택

Python · MySQL · BeautifulSoup · Requests

Passenger Survival Prediction Model

프로젝트 설명

타이타닉 승객 CSV 데이터를 기반으로 주요 변수를 정제하고, 생존 여부 예측 모델을 학습해 입력 데이터에 대한 예측 결과를 확인할 수 있도록 구성한 프로젝트입니다. 데이터 전처리, 학습 데이터 구성, 예측 결과 도출까지의 흐름을 직접 구현했습니다.

주요 내용
  • CSV 데이터 정제 및 결측치 처리
  • 학습용 데이터 전처리 및 모델 구성
  • 입력값 기반 생존 확률 예측 기능 구현
  • 데이터 처리부터 예측 결과 도출까지 전체 흐름 구성
기술 스택
